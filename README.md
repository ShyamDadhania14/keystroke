## 🔥 what is this thing

A typing speed test that got bored of being polite and decided to start a fight instead.

Most typing tests just... sit there. You type, you get a number, you close the tab. **Keystroke makes it personal.** You're not typing into the void — you're racing a 🦊 fox, a 🤖 cyborg, or worse: **your own ghost from your best run ever**, replaying stroke-for-stroke right next to you.

> *"I just wanted to check my WPM. It's 2am. I have raced the Lightning bot eleven times. I refuse to lose to lightning."*
> — every single person who opens this site

## 🎮 how to play

```
1. Pick a mode      →  Quotes / Words / Code / Punctuation
2. Pick a time       →  15s / 30s / 60s / 120s
3. Pick an opponent  →  Turtle, Rival, Cyborg, Lightning, or your own Ghost
4. Click the box     →  start typing immediately
5. Don't lose        →  seriously, don't
```

**Keyboard shortcuts** (because real typists never touch the mouse):

| Key | Action |
|---|---|
| `Tab` | Restart instantly |
| `Esc` → `Enter` | Restart (the "I rage quit but I'm back" combo) |
| `Backspace` | Fix your mistake before the bot sees it |

## 🛠️ tech stack

**Vanilla HTML, CSS, and JavaScript. That's it. That's the whole stack.**

No frameworks. No build step. No `node_modules` folder eating 400MB of your hard drive. No backend, no database, no signup wall. Just three files that do exactly what they say:

```
keystroke/
├── index.html     →  structure
├── style.css      →  the dark-mode glow-up
└── script.js      →  the actual game brain
```

Open `index.html` in any browser. That's the install process. You're welcome.

## 🧠 the science (sort of)

| Metric | What it actually means |
|---|---|
| **Net WPM** | Your real speed — correct words only, mistakes don't count |
| **Raw WPM** | Your speed including the chaos — every keystroke, right or wrong |
| **Accuracy** | Correct keystrokes ÷ total keystrokes. Be honest with yourself. |
| **Combo** | Consecutive correct keystrokes. Resets to zero the second you slip. |

The average person types **40 WPM**. Professional typists hit **65–75 WPM**. Competitive speed typists break **120+ WPM**.

The Lightning bot types at **120 WPM.**

You see the problem.
